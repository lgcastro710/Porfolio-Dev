■ Frontend

Riky & Morty

Aplicación web desarrollada con React que consume la API oficial de Rick and Morty para explorar personajes de la serie de manera rápida e intuitiva. Permite buscar personajes por nombre, aplicar filtros, navegar mediante paginación y acceder a información detallada de cada personaje. La aplicación incluye un sistema de favoritos para guardar personajes preferidos, un selector de tema oscuro y claro (Dark/Light Mode) para personalizar la experiencia visual, y un diseño completamente responsive adaptado a dispositivos móviles, tablets y escritorio.
